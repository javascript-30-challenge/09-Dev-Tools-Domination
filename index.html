Console Methods

Using console methods can be great for manipulating your data, application or routes to show you tons of different information about what is going on.

Here are the top 13 console methods!!

1. console.log()

Ex: console.log(variableName) or console.log(object)
This results in showing the data in your console.

2. String Interpolation

Ex: console.log('Hello I am a %s string!', 'user')
This will result in putting 'user' in place of the '%s'.
We also can use the newer method using ${} which is an easier method.

3. Styling your console

Ex: console.log('%cI am some great text', 'font-size:20px; background:blue')
This will result in your text being 20px in size and the color blue in your console.

4. console.warn()

Ex: console.warn('This code only works when you make it camelcase')
This will result in a warning logged in the console with the text you added.

5. console.error()

Ex: console.error(error)
This will log an error to the console if something goes wrong as it will trigger the error.

6. console.info()

Ex: console.info('If you are refactoring this code make sure you keep the variable names')
This will result in an info block logged to the console showing you the info that was inputted.

7. console.assert()

Ex: const p = document.querySelector('p')
console.assert(p.classList.contains('ouch'), 'That is wrong!')
This will result in an error with the included text being logged out to the console if the criteria is false.

8. console.clear()

Ex: You have a lot of warnings or information in your console. Running 'console.clear()' will clear all that information out.

9. console.dir()

Ex: console.dir(p)
This will result in logging the p element with a drop down of all the attributes of the element.

10. console.group()

Ex: dogs.forEach(dog => {
console.groupCollapsed(`${dog.name}`) or console.group(`${dog.name}`)
console.log(`This is ${dog.name}`)
console.log(`${dog.name} is ${dog.age} years old`)
console.log(`${dog.name} is ${dog.age * 7} dog years old`)
console.group(`${dog.name}`)
})

11. console.count()

Ex: console.count('wes')
console.count('wes')
console.count('Steve')
console.count('wes')
console.count('Steve')
console.count('Steve')
console.count('Steve')
console.count('Steve')
console.count('Steve')
console.count('wes')
console.count('Steve')
This will result in a constant count of the same elemnts logged to the console

12. console.time()

Ex: console.time('fetching data')
fetch('https://api.github.com/users/wesbos')
.then(response => response.json())
.then(data => {
console.timeEnd('fetching data')
console.log(data)
})
This would result in logging the time it took to complete the fetch request and the data being returned.

13. console.table()

Ex: console.table(fetchData)
This would result in logging a table to the console that neatly organizes and lays out the information being returned.
This method is a great way to visualize the json data being returned from making an API request.
